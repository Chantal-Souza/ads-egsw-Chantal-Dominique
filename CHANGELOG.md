# Changelog

## [1.2] - 10-11-2025

### Adicionado
- Sistema de busca de paciente por nome
- Sistema de busca de médico por nome 
- Exibição de dados completos e agendamentos relacionados nas buscas
- Nova opção no menu principal para funcionalidades de busca

### Alterado
- Menu principal expandido para incluir opção de buscas
- Estrutura do sistema para suportar funcionalidades de consulta avançada

## [1.1] - 09-11-2025

### Adicionado
- Menu "Visualizar cadastros" no menu principal
- Submenu "Lista de Cadastros" com 5 opções:
  - Pacientes Cadastrados
  - Atendentes Cadastrados  
  - Médicos Cadastrados
  - Consultas Agendadas
  - Voltar ao Menu Principal
- Sistema de listagem completa de todos os dados cadastrados
- Interface para navegação entre telas de consulta
- Funcionalidade de retorno ao menu principal

### Alterado
- Menu principal expandido para 6 opções
- Organização do código para suportar funcionalidades de consulta

## [1.0] - 08-11-2025
### Adicionado
- Classes básicas: Paciente, Funcionario, Atendente, Medico, Consulta
- Menu inicial com 4 opções principais
- Sistema de cadastro e agendamento
- Dados de exemplo para testes
